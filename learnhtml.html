What is Python?

Python is a high-level, interpreted programming language known for its readability and simplicity. It supports multiple paradigms: procedural, object-oriented, and functional programming.

1. Variables & Data Types

• Variables are names bound to values. You don’t declare type explicitly: 

age = 30
name = "Alice"
pi = 3.14159
is_active = True

• Built-in Types:

  • int (integers)
  • float (floating-point numbers)
  • str (strings)
  • bool (True/False)
  • list, tuple, dict, set

2. Control Flow

• If / Elif / Else:

score = 85
if score >= 90:
    print("A")
elif score >= 80:
    print("B")
else:
    print("C")

• Loops:

# For loop over a list
for item in [1, 2, 3]:
    print(item)

# While loop
count = 0
while count < 3:
    print(count)
    count += 1

3. Functions

• تعرف الدوال باستخدام def، ويمكنها إرجاع قيم باستخدام return: 

def greet(name):
    """Prints a greeting."""
    return f"Hello, {name}!"

message = greet("Bob")
print(message)  # Hello, Bob!

• يمكن للدوال أخذ معلمات افتراضية (default arguments): 

def power(base, exp=2):
    return base ** exp

print(power(5))     # 25
print(power(5, 3))  # 125

4. Classes & OOP

• تستعمل الفئة (class) لتعريف كائنات مع خصائص وسلوك: 

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def say_hello(self):
        print(f"Hi, I'm {self.name} and I'm {self.age}.")

p = Person("Eve", 28)
p.say_hello()  # Hi, I'm Eve and I'm 28.

• Inheritance لوراثة الخصائص من صنف آخر: 

class Student(Person):
    def __init__(self, name, age, major):
        super().__init__(name, age)
        self.major = major

5. Modules & Packages

• لتنظيم الكود نقسمه إلى ملفات (.py) ونستوردها: 

# In math_utils.py
def add(a, b):
    return a + b

# In main.py
from math_utils import add
print(add(2, 3))  # 5

• لإنشاء حزمة ضع ملف __init__.py داخل المجلد.

6. Exceptions

• استخدم try/except للتعامل مع الأخطاء: 

try:
    x = int("abc")
except ValueError as e:
    print("Conversion failed:", e)
finally:
    print("Done.")

7. File Handling

• لقراءة/كتابة الملفات استخدم: 

with open("data.txt", "r") as file:
    content = file.read()

with open("out.txt", "w") as file:
    file.write("Hello, file!")

8. List Comprehensions

• صيغة مختصرة لإنشاء قوائم: 

squares = [x**2 for x in range(10)]
evens = [x for x in range(10) if x % 2 == 0]
