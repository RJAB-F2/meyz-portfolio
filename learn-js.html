What is JavaScript?

JavaScript is a programming language used to make web pages interactive. You can use it to respond to user actions, update content dynamically, validate forms, create animations, and more.

How to Use JavaScript

  • Inline: Add directly to HTML elements using the onclick or other event attributes.
  • Internal: Write inside a <script> tag within the HTML file.
  • External: Link a separate .js file using <script src="script.js">.

Basic Example

<button onclick="greet()">Click me</button>

<script>
  function greet() {
    alert("Hello, welcome to JavaScript!");
  }
</script>

JavaScript Concepts

  • Variables: let, const, var
  • Functions: Blocks of code you can reuse
  • Events: Actions that trigger scripts (like clicks)
  • DOM Manipulation: Changing HTML content dynamically
