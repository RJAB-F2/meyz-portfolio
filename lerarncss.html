What is CSS?

CSS (Cascading Style Sheets) is the language used to style web pages. It allows you to control colors, fonts, layouts, spacing, animations, and more.

How to Use CSS?

You can apply CSS in three main ways:

  • Inline: Directly in the HTML element using the style attribute.
  • Internal: Inside a <style> tag in the HTML <head>.
  • External: Using a separate file like style.css and linking it via <link>.

Basic Example

body {
  background-color: black;
  color: white;
  font-family: Arial;
}

h1 {
  color: #00d8ff;
  text-align: center;
}

CSS Selectors

  • Element Selector: targets HTML tags like p, h1
  • Class Selector: uses a dot . like .box
  • ID Selector: uses a hash # like #main
